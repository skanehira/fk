# fk
Kill specific processes with fuzzy finder.

![](https://i.imgur.com/Q6ONFRz.gif)

## Usage
```sh
$ fk
```

Keybindings

| key    | description     |
|--------|-----------------|
| CTRL-i | select/unselect |
| CTRL-j | go to next      |
| CTRL-k | go to prev      |
| CTRL-c | abort           |
| Enter  | kill            |

## Instalttion
```sh
$ go get github.com/skanehira/fk/cmd/fk
```

or Download from [releases](https://github.com/skanehira/fk/releases)
